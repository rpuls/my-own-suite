# My Own Suite Site

This folder contains the Astro + Starlight website for `myownsuite.org`.

## Local development

```bash
cd site
npm i
npm run dev
```

## Build and preview

```bash
npm run build
npm run preview
```

## Cloudflare Pages

Use these settings:

- Build command: `npm run build`
- Output directory: `dist`
- Node version: `20` (recommended)
