My Own Suite is a curated collection of open-source tools that can replace the everyday online services many people use from Google, Microsoft, and Apple.

We have picked the tools and bundled them together so you do not have to piece everything together yourself.

My Own Suite is designed to run in the cloud so it can be available from all your devices, wherever you are. That is what makes private file sync, photo backup, password access, calendars, sharing, and browser-based tools possible.

It can also run fully locally and offline for free if cloud features are not what you want. But if you want a real alternative to Google, Microsoft, or Apple across devices, cloud access is a big part of the experience.

In short: My Own Suite is your private cloud.

If you are still unsure why that is different from using Google, Microsoft, or Apple cloud products, read [Why your own cloud?](/docs/why-your-own-cloud).

## Is My Own Suite free?

The software in My Own Suite is free and open source.

If you run it locally on your own machine, you can use it for free.

If you want cloud access across devices, then it needs somewhere to run. That is where cost can come in:

- Railway or a VPS means paying for hosted infrastructure
- your own hardware means using equipment, power, and your own setup time

So the stack itself is free. The possible cost is where and how you choose to run it.

## Included apps

The apps in My Own Suite were picked because they are trusted, widely loved, and already proven in the real world, while still offering the kind of smooth everyday experience people usually expect from mainstream products.
